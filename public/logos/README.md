# Logos Folder

Add your tool logos here. Supported formats: `.svg`, `.png`, `.jpg`, `.webp`

## Current logos configured:
- n8n.svg
- openai.svg
- anthropic.svg
- langchain.svg
- react.svg
- typescript.svg

## How to add logos:
1. Add your logo files to this folder (`public/logos/`)
2. Update the logo array in `src/components/sections/Hero.tsx` to include your logo filename
3. Logos will automatically display in the Hero section

## Logo specifications:
- Recommended size: 100-200px height
- Format: SVG preferred (scalable), PNG with transparent background also works
- The logos will be displayed in grayscale by default and become colored on hover
